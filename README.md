# xtrlock-to
Wrapper for [xtrlock](https://packages.debian.org/sid/xtrlock).

## Usage

```text
Usage: ./xtrlock-to.pl options:
  -t=minutes (timeout)
  -b (blank screen after lock)
  -d (run as daemon)
```
